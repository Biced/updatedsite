Architecture & Design

A Wordpress website.
The client asked for the site to be modern responsive and showcase high-quality pictures.

The things I had to customize were:
the Accordion navigation menu, Accessibility, mobile burger menu animation. everything else was done with Elementor pro from scratch and the skelementor theme to remove all the WordPress defaults.

The design:
This project contains a lot fo high-quality pictures the challenge was finding the right balance of white space The logo is just a font with an Html span for the color The client is a B to B freelance, therefor conveying trust was important and we picked blue as the main color.
