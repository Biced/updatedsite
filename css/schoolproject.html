תרגיל בית – שימוש באחסון נתונים

מטרת התרגיל – שימוש בתצורות של אחסון נתונים ואסינכרוני

עליך ליצור מספר דפים להלן - שים לב יש לבצע שימוש ב Bootstrap יחד עם שימוש ב CSS 3 וב HTML 5 כדי שהדפים יהיו ריספונסיביים.

להלן רשימת הדפים:

דף רישום סטודנט שיכיל שם משתמש וססמה. את שם המשתמש והססמה עליך לשמור בתצורה של שימוש ב-localStorage דף כניסה לאחר שימוש בשם המשתמש והססמה מתוך ה-localStorage.
לאחר כניסת הסטודנט לדף זה, יופיע לסטודנט דף רישום לקורס. עליך לייצור את כל השדות ולשמור את המידע תוך שימוש ב-storage. לאחר הופעת הדף, עליך להסתיר את כפתור "הירשם" ולהציג כפתורי עדכון ומחיקה. את הפונקציות לרישום, עדכון ומחיקה יש לכתוב בצורה אסינכרונית. כמו כן תהיה אפשרות לעדכון המידע של אחד מהשדות שהוכנסו. ובנוסף תהיה אפשרות של מחיקה של כל שדה

שאלת אתגר – כעת לאחר שסיימת, יש לך את פרטי הסטודנט ואת פרטי הקורס שאליו הסטודנט רוצה להירשם, הצג את כל הפרטים בצורה טבלאית מעוצבת בדף נוסף. בדף זה עליך לייצר אייקון שיחזיר את המשתמש לדף הבית וינקה את כל אחסון הנתונים.
